## 🌾 JASteria とは？

**JASteria（ジャステリア）** は、有機農林規格 **「JAS 0004（有機料理を提供する飲食店）」** の認証基準に完全適合した、ペーパーレス監査＆ロットトレーサビリティ管理システムです。

伝統ある格付マークの信頼性と、モダンなレストラン実務を調和させ、監査準備の負担をゼロにすることを目指して設計されています。大地の恵みを感じさせる「オリーブグリーン × 木目アンバー」のプレミアムな和調美装デザインを採用。お洒落なレストランのiPadやPC端末に美しく溶け込みます。

---

## ✨ 主要機能

### 1. 🔍 JAS審査官ポータル ＆ 3点連動ロット追跡
*   **逆引きトレーサビリティ**: 調理提供（売上実績）から、製造調理点検（レシピ・ロット）、そして仕入受入点検（JASマーク写真、仕入証跡）へ1秒で逆引き追跡。
*   **マテリアルバランス自動照合**: 提供食数に合わせた有機原材料の必要理論値を自動計算。仕入受入量との整合性を数学的に証明し、すり替えや不正混入がないことを一瞬で可視化します。
*   **台帳印刷出力**: JAS認証機関の審査員に提出できる「JAS 0004 トレーサビリティ検証確認書」のフォーマット印刷。

### 2. 🧹 衛生・清掃点検記録簿（JAS 0004完全適合）
*   **確実な衛生エビデンス**: 床、壁、調理台の清掃・消毒（アルコール等）、ゴミの分別処理、非有機一般品との接触混同防止チェックリストを完備。
*   **過去指摘事項の対策**: 過去のJAS監査における「ホールの清掃記録」などの指摘改善に完全適合した仕様。

### 3. 📷 JASマーク現物写真保管（実物エビデンス）
*   **スマホダイレクト保管**: 納品時にその場でJASマーク実物を撮影し、受入台帳にロット番号と紐付けて永続保存。2025年人参等の「JASマーク現物確認」の指摘に対応。
*   **自動生成デモスタンプ**: 監査シミュレーションや練習用に、美しいフォレストグリーンの「有機JAS検証スタンプ（SVG）」をワンクリックで自動貼付。

### 4. 🍶 料理酒JAS義務化リアルタイム警告
*   **令和7年10月1日法改正対応**: レシピに一般料理酒が含まれるメニューに対し、JAS割合（95%）にかかわらず「有機料理酒の使用が必須化される」旨の動的警告をアラート。早期の切り替え準備をアシスト。
